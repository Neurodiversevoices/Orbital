## ARTIFACTS — QCR (Quarterly Capacity Report)

QCRs are one-time purchases. Scope must match account type.

| Scope | Price | Requirements |
|-------|-------|--------------|
| Individual QCR | $149 | Requires Individual subscription |
| Circle QCR | $299 | Requires Circle subscription |
| Bundle QCR | $499 | Requires Bundle subscription + Admin role |

### QCR Features by Scope

**Individual QCR ($149)**
- Covers one person only
- 90-day pattern synthesis
- Resilience metrics
- Recovery velocity analysis
- Clinical-grade PDF export
- EHR-attachment ready

**Circle QCR ($299)**
- Covers entire Circle (up to 5 people)
- Relational dynamics analysis
- Synchronization patterns
- Load distribution insights
- Individual + aggregate views

**Bundle QCR ($499)**
- Program-level analysis
- Admin-facing insights
- Aggregate capacity trends
- Intervention timing recommendations
- Deterministic templates only (no open-ended AI)

## CLASS B — INSTITUTIONAL ("CITADEL")

Class B is a **Risk Transfer Protocol**, not SaaS pricing.

Contract-only. No self-serve. No bundled pricing.

---

### Layer 1 — Sovereignty License

| Component | Price |
|-----------|-------|
| Annual Fee | $250,000/year (flat) |

**What it provides:**
- Private container deployment
- Domain lock enforcement
- SSO integration
- Rule-of-5 firewall (K-anonymity)
- Legal boundary documentation

**What it does NOT include:**
- 0 users
- 0 dashboards
- 0 visibility rights

**Budget Owner:** IT / SecOps

---

### Layer 2 — Capacity Coverage

| Component | Price |
|-----------|-------|
| Per-Employee Fee | $1,000/employee/year |

**What it provides:**
- Right to ingest capacity signals from covered employees
- Signal storage within sovereign container
- Aggregation eligibility

**What it does NOT include:**
- No visibility rights
- No governance rights
- No dashboards

**Critical Principle:** Collection ≠ Visibility

**Budget Owner:** HR / Benefits / Insurance

---

### Layer 3 — Governance & Intelligence Rights

| Component | Price |
|-----------|-------|
| Annual Fee | $150,000/year (flat) |

**What it provides:**
- Triage Command Center access
- Governed aggregation (labor-safe abstraction)
- K-anonymity enforced views
- Cached dashboard access (no real-time)
- Export rights (aggregate only)

**Critical Principle:** This layer operationalizes data. Without it, collected signals are intentionally non-actionable.

**Budget Owner:** Legal / Risk

---

### Layer 4 — Strategic Assurance

| Component | Price |
|-----------|-------|
| Annual Fee | $100,000/year (flat) |

**What it provides:**
- Quarterly Operational Risk Statements
- Board-level artifact, signed by Orbital
- Fiduciary documentation for directors
- Audit-ready capacity posture reports

**Budget Owner:** CEO / Board

### Class B Structural Constraints

- Contract-only (no self-serve)
- No bundled enterprise pricing
- No per-seat SaaS model
- Anonymous aggregates only (Rule of 5 enforced)
- No named individuals
- No individual timelines
- Cached dashboards only (no real-time)
- Domain gatekeeping blocks enterprise domains from Class A
- **Visibility requires Governance Layer (Layer 3)**
