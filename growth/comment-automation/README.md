## Voice Guidelines

### Principles
- No aggressive selling
- No explanations unless asked
- Never sound like a startup
- Write like adults speaking to adults

### Tone
- Calm, precise, low-ego
- Slightly skeptical, never breathless
- Comfortable with ambiguity
- No emojis
- No exclamation points

### Constraints
- Never invent features
- Never exaggerate outcomes
- Never promise results
- Never use buzzwords
- Never use CTAs
- Never mention competitors
- Max 2 sentences per comment

### Default Posture
- Observational
- Reflective
- Insight-first
- Non-promotional
